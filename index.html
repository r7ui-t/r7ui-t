01 / ABOUT ME

高校時代に学んだ3つのこと

最初の一歩は環境づくり

高校入学前に自分用の PC を購入。道具を先に整えたことで、調べる・作る・壊すを毎日繰り返せる土台ができました。

合わない道を見極める勇気

見よう見まねでフロントエンド制作を続けたものの、うまくいかず挫折しかけました。そこで「自分が本当に学びたいのは別領域かもしれない」と考え、言語選択を見直しました。

完成させる体験の価値

Python で作ったシンプルなチャットボットを友達と遊べる形で完成。規模が小さくても最後まで作る経験が、自信と次の学習意欲につながりました。

02 / WORKS

高校時代の主なアウトプット

HTML / CSS / JS

簡単なマリオ風ゲーム

プログラミングに複数言語があると知り、最初に選んだのは HTML/CSS/JS。基礎学習後、ブラウザで動くシンプルな横スクロールゲームを制作しました。

  • 内容: キャラクター移動、ジャンプ、障害物判定
  • 課題: 見よう見まね実装では再現性が低い
  • 学び: 仕組みを理解して書く重要性

Python / Solo

シンプルチャットボット

フロントエンド中心の学習から方向転換し、Python の基礎を学習。条件分岐と辞書を使った簡単なチャットボットを作成しました。

  • 内容: あいさつ、簡易応答、終了コマンド
  • 成果: 友達同士で実際に遊べる形で公開
  • 学び: 小さくても完成させると次に進める

Django / Web App

動きのあるWebアプリ試作

Python を活かして Web アプリ開発に進みたくなり、Django を学習。フォーム送信とデータ表示を備えた小規模アプリを試作しました。

  • 内容: 画面遷移、入力、一覧表示の基本実装
  • 課題: 設計とDB理解の不足を実感
  • 学び: バックエンド知識の必要性を認識
